# Randomware demo script (2:30)

**0:00–0:20 — framing.** “Randomware is a slot machine for software. A human wrote the product direction; GPT-5.6 Sol designed the safety contract, and GPT-5.6 Luna implemented one goal in Codex under a $100 credit budget.”

**0:20–0:45 — spin.** “I spin two bounded public APIs. The reels reveal their names, categories, and roles. The model gets a collision, not a template.”

**0:45–1:20 — concept and build.** “The model proposes an eccentric but structured concept, then submits one complete single-file app. Randomware validates the byte range, markers, and literal broker calls before publication.”

**1:20–1:55 — live result.** “The owner page keeps the warning and attribution outside the sandbox. The generated app is opaque-origin, and its button calls only the companion broker. I can open Source and Requests to see the exact accepted revision and mediated traffic.”

**1:55–2:15 — second result.** “I spin again or mutate the same APIs. The collision changes while the selected operation set stays bounded. A failure is an honest death certificate, not a fake success.”

**2:15–2:30 — close.** “The design came from Sol, the implementation from Luna through Codex, and the final judgment stays with the human. Real APIs go in; random apps — or inspectable failures — come out.”
